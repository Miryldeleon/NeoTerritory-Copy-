# Trees

- Folder: `docs/Codebase/Microservice/Modules/Header/Trees`
- Role: contracts for actual, virtual, broken, and shared tree structures

## Read Order
1. `Actual/`
2. `Broken/`
3. `Shared/`

## Boundary
- `Actual/` holds the primary parse-tree contracts and internal helpers.
- `Broken/` mirrors broken-tree contracts used by pattern-specific views.
- `Shared/` holds contracts reused by more than one tree subtype.
